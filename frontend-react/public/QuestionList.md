# 탐색적 데이터 분석 (EDA)

## 기본 통계
* 데이터의 기본 통계 요약을 보여줘
* 각 변수의 분포를 시각화해줘

## 이상치 및 패턴
* 결측 컬럼 제거해줘
* 이상치를 탐지하고 시각화해줘
* 변수 간 상관관계를 분석해줘
* 데이터의 서브셋 패턴을 찾아줘

## 시각화
* 결측맵 그려줘

# 데이터 전처리

## 클리닝
* 결측값을 처리해줘
* 이상치를 제거하거나 대체해줘
* 중복 데이터를 제거해줘

## 변환
* 범주형 변수를 인코딩해줘
* 수치형 변수를 정규화해줘
* 새로운 피처를 생성해줘
* 타겟과 설정된 변수들만으로 데이터 프레임 새로 구성해줘

# 피처 선택

* 핵심 인자를 추출해줘
* 인자를 최소화해줘
* Null Importance 기반 피처 선택을 수행해줘
* 주요 변수 중요도를 시각화해줘

# 모델링

## 회귀 모델
* 회귀 모델을 학습시켜줘
* 여러 회귀 모델을 비교해줘
* 최적 하이퍼파라미터를 탐색해줘

## 모델 평가
* 모델 성능을 평가해줘
* 잔차 분석을 수행해줘
* 교차 검증 결과를 보여줘

# 모델 해석

* SHAP 값으로 피처 기여도를 분석해줘
* 개별 예측에 대한 SHAP 설명을 보여줘
* 주요 인자의 영향력을 시각화해줘

# 최적화

* 목표값을 최대화하는 입력 조건을 찾아줘
* 목표값을 최소화하는 입력 조건을 찾아줘
* 특정 제약 조건 하에서 최적 조건을 탐색해줘
